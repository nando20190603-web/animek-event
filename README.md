# ANIMEK イベント管理アプリ

コスプレイベント向けのリアルタイム参加者管理 Web アプリです。  
サーバー不要のシングルファイル構成で、GitHub Pages から即座に公開・利用できます。

**公開 URL:** https://nando20190603-web.github.io/animek-event/

---

## 機能一覧

### 参加者（コスプレイヤー）
| 機能 | 説明 |
|------|------|
| コスプレイヤー一覧 | 登録済みコスプレイヤーをキャラ・作品・エリアで検索・フィルター |
| 会場マップ | Google Maps で会場ピン・現在地を表示 |
| コスプレ登録 | キャラ名・作品名・撮影可否・現在エリアを登録 |
| チェックイン | GPS で会場に近づいた際にチェックイン |
| 撮影依頼 | 撮影 OK のコスプレイヤーにカメラマンへの撮影依頼を送信 |

### カメラマン
| 機能 | 説明 |
|------|------|
| 撮影依頼一覧 | コスプレイヤーからの依頼を一覧表示 |
| 依頼ステータス管理 | 対応中 / 撮影完了 / お断り を更新 |

### 管理者
| 機能 | 説明 |
|------|------|
| ダッシュボード | 登録数・チェックイン数・会場混雑状況をリアルタイム表示 |
| 参加者管理 | コスプレイヤーの確認・削除 |
| 会場管理 | 会場の追加・編集・地図ピッカーで座標設定 |
| チェックイン管理 | 会場ごとの入退場を手動管理 |
| イベント設定 | イベント名・開催日・参加コード・カメラマンコードなど |

アクセス URL を開くとログイン画面が表示されます。

| ロール | タブ | 初期コード / パスワード |
|--------|------|------------------------|
| 参加者 | 参加者 | `ANIMEK2025` |
| カメラマン | カメラマン | `PHOTO2025` |
| 管理者 | 管理者 | `animek2025` |

> コードとパスワードはすべて管理者設定画面から変更できます。

## 初期設定（管理者）

1. 管理者としてログイン
2. **設定** 画面を開く
3. 以下を設定して「保存」

| 項目 | 説明 |
|------|------|
| イベント名 | 表示名 |
| 開催日・開場時間 | 参加者のプロフィール画面に表示 |
| 参加コード | 参加者ログイン用コード |
| カメラマンコード | カメラマンログイン用コード |
| Google Maps API キー | 地図表示に必要（後述） |
| チェックイン GPS 半径 | 会場からこの距離（m）以内でチェックイン可 |
| 管理者パスワード | 初期値から変更推奨 |

## Google Maps の設定

地図機能を使うには Google Maps JavaScript API キーが必要です。

1. [Google Cloud Console](https://console.cloud.google.com/) でプロジェクトを作成
2. **Maps JavaScript API** を有効化
3. API キーを作成し、**HTTP リファラー制限** に以下を追加：
   ```
   https://nando20190603-web.github.io/*
   ```
4. 管理者設定画面の「Google Maps API キー」に貼り付けて保存

---

## データの保存について

すべてのデータはブラウザの **localStorage** に保存されます。

- サーバーは不要（バックエンドなし）
- **同一デバイス・同一ブラウザ** 内でデータが共有されます
- 別デバイスとのリアルタイム同期はされません（「更新」ボタンで再読み込み）
- ブラウザのデータ消去でデータが削除されます

| キー | 内容 |
|------|------|
| `animek_cosplayers` | コスプレイヤー一覧 |
| `animek_venues` | 会場一覧 |
| `animek_photo_requests` | 撮影依頼一覧 |
| `animek_activity` | アクティビティログ |
| `animek_settings` | イベント設定 |
| `animek_maps_key` | Google Maps API キー |
| `animek_admin_pw` | 管理者パスワード（ハッシュなし・要注意） |

## ファイル構成

```
animek-event/
├── index.html      # アプリ本体（HTML / CSS / JS すべて単一ファイル）
├── sw.js           # Service Worker（キャッシュ対策・オフライン対応）
├── refresh.html    # キャッシュ強制更新用ページ
├── .nojekyll       # GitHub Pages の Jekyll 処理を無効化
└── README.md       # このファイル
```

---

## キャッシュが古い場合

デバイスで古いバージョンが表示される場合は、以下の URL にアクセスしてください：

```
https://nando20190603-web.github.io/animek-event/refresh.html
```

アクセスするとキャッシュをクリアして最新版にリダイレクトします。

```bash
git add .

git push origin main
```

GitHub Pages（`main` ブランチ）に自動反映されます（反映まで数分かかる場合あります）。
